# nand2tetris-solution

Kumpulan jawaban dan artefak pengerjaan course:

1. Build a Modern Computer from First Principles: From Nand to Tetris (Project-Centered Course)  
   https://www.coursera.org/learn/build-a-computer
2. Build a Modern Computer from First Principles: Nand to Tetris Part II (project-centered course)  
   https://www.coursera.org/learn/nand2tetris2

## Struktur Repo

- `training/01` sampai `training/06`: jawaban per project (HDL/ASM/HACK sesuai kebutuhan project)
- `training/*/report`: hasil test dan ZIP submission
- `docs/`: catatan dan penjelasan tambahan
