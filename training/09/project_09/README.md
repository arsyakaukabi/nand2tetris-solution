# Project 09 - Flappy Bird (Jack)

## Submission Metadata (Coursera)

- Project Title: `FlappyBird`
- Title: `FlappyBird`
- Caption: `A simple Flappy Bird clone in Jack. Press any key to start and flap, avoid moving pipes, and survive as long as possible while your score increases.`

Folder game:
- `FlappyBird/Main.jack`
- `FlappyBird/Game.jack`
- `FlappyBird/Bird.jack`
- `FlappyBird/Pipe.jack`

## Cara compile

Dari root repo:

```bash
nand2tetris compile training/09/project_09/FlappyBird
```

## Cara run (VM Emulator / web-ide)

Load folder `training/09/project_09/FlappyBird` pada VM Emulator, lalu jalankan.

Kontrol:
- Tekan tombol apa pun untuk start
- Saat game berjalan, tekan tombol apa pun untuk flap
- Saat game over, tekan tombol apa pun untuk restart
